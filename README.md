# Sistema de Reserva de Salas FIAP

## Descrição do Problema
Dificuldade de encontrar salas disponíveis para estudo.

## Solução Proposta
Sistema em Python que permite reservar salas de estudo.

## Tecnologias
- Python
- JSON

## Como Executar
python sistema_reserva_salas.py

## Funcionalidades
- Ver salas
- Reservar
- Cancelar
- Listar reservas
